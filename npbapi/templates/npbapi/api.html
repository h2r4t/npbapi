試合速報

リクエストURL

https://npbapi-147514.appspot.com/score/today

更新

レギュラーシーズン中は毎日12:00〜23:59まで約60秒毎に情報を更新します。
オープン戦・オールスター・クライマックスシリーズ・日本シリーズは非対応です。

レスポンス
フィールド 内容 説明
inning イニング b:表, t:裏, before:試合前, after:試合終了, canceled:試合中止
home_team ホームチーム名 H, F, M, L, Bs, E, S, G, T, C, D, DB
home_score ホームチーム得点 試合前は「-」
visitor_team ビジターチーム名 H, F, M, L, Bs, E, S, G, T, C, D, DB
visitor_score ビジターチーム得点 試合前は「-」
start_at 試合開始時間 例) 18:00
